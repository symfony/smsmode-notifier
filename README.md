Smsmode Notifier
================

Provides [Smsmode](https://www.smsmode.com/) integration for Symfony Notifier.

DSN example
-----------

```
SMSMODE_DSN=smsmode://API_KEY@default?from=FROM
```

where:

 - `API_KEY` is your Smsmode API key
 - `FROM` is your sender ID
